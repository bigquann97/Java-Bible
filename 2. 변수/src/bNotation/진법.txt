3. 진법
3.1 10진법과 2진법
주로 10진법 사용 - 컴퓨터는 2진법사용
2진수밖에 모르는 컴퓨터 - 많은 자리수가 필요

3.2 비트(bit)와 바이트(byte)
한자리의 2진수 - bit - 컴퓨터가 값을 저장할 수 있는 최소단위
8bit = 1byte // 4byte = word(CPU가 한번에 처리할 수 있는 데이터의 크기 - CPU 성능에따라 달라짐)
32비트 CPU에서 1워드는 32bits(4bytes), 64비트 CPU에서 1워드는 64bits(8bytes)
n비트로는 2의n승개의 값을 표현 가능 - n비트로 10진수를 표현한다면 범위는 0 ~ 2의n승 -1까지 표현 가능 - 4bits는 0 ~ 15까지 표현 가능

3.3 8진법과 16진법
8진수는 2진수 3자리를, 16진수는 2진수 4자리를 각각 한자리로 표현 - 자리수 짧아져 알기 쉽고 서로 형변환 간단

2진수를 8진수, 16진수로 변환
2진수 => 8진수 - 2진수를 뒤에서부터 3자리씩 끊어 해당하는 8진수로 변환
2진수 => 16진수 - 위와 같이 4자리씩 끊기

3.4 정수의 진법 변환
10진수를 n진수로 변환 - 해당 진수로 나누고 나머지 값을 옆에 적는것을 나눌 수 없을때까지 반복 - 마지막 몫과 나머지를 아래부터 위 순서대로 적기
n진수를 10진수로 변환 - n진수의 자릿수는 n의 m승으로 표현해 해당 자리수별로 곱해 더한 값

3.5 실수의 진법 변환
10진 소수점수를 2진 소수점수로 변환하는 방법
 1. 10진 소수에 2를 곱한다 // 2. 위의 결과에서 소수부만 가져다 다시 2를 곱한다 // 3. 1, 2 과정을 소수부가 0이 될때까지 반복한다
 => 위의 결과에서 정수부만을 위에서 아래로 순서대로 적고 0.을 앞에 붙인다
2진 소수점수를 10진 소수점수로 변환하는 방법
 2의 -n승의 값을 각 자릿수에 곱해 더해준다

3.6 음수의 2진 표현 - 2의 보수법
가장 왼쪽의 첫 번째 비트를 MSB - Most Significant Bit - 로 설정해 부호비트로 설정
0~7, -0~-7을 2진수 순서대로 표현시 0이 두개 존재하며, 2진수가 증가할때 10진 음수는 감소하는 형태가 됨
따라서, 2의 보수법에 의해 음수를 배치 - 절대값이 같은 양수와 음수를 더했을때 2진수로도 0을 결과로얻어 부호신경 안써도됨
+ 2진수가 증가할 때 10진 음수가 감소한다는 모순도 없어짐

2의 보수법
어떤 수의 n의 보수는 더했을때 n 이 되는 수 => 2의 보수관계에 있는 두 2진수를 더하면 자리올림이 발생하고 0이 됨

음수를 2진수로 표현하기
10진 음의 정수를 2진수로 변환하려면, 10진 음의 정수의 절대값을 2진수로 변환 후, 2의 보수를 구한다
-5의 절대값 5 => 5의 2진수 0101 => 0101의 2의 보수 1011

2의 보수 구하기 - 2의 보수 = 1의 보수 + 1
1의 보수는 0을 1로 1을 0으로 바꾸기
